## Overview
The goal of this lab is to move away from "binary" design—where actions are instant and permanent—toward Forgiving Design. You will apply Human-Computer Interaction (HCI) theories to create a system that assumes the user is human and will eventually make a mistake.

Submission Requirement: You are required to submit a link to a working UI prototype (e.g., Figma prototype link, Adobe XD, Gemini Link or a hosted web link). Do not submit code.

## Your Objective
Redesign a social media "Delete Account/Post" process using three specific HCI "safety nets" to bridge the gap between user intent and system action.
